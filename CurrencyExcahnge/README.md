# CurrencyExchange

Консольное приложение, которое получает курсы USD → RUB/EUR/GBP через публичный API и демонстрирует, как правильно настраивать отказоустойчивость HTTP-клиента через **Polly**.

## Что происходит при запуске

```text
1 USD = 88.50 RUB
1 USD = 0.92 EUR
1 USD = 0.78 GBP
```

Если сеть недоступна или API не отвечает - приложение сообщает об этом понятным сообщением, не стектрейсом.

## Про Polly

Три политики навешаны на `HttpClient` через `.AddPolicyHandler`:

**Retry** - при временной ошибке или таймауте повторяет запрос до 3 раз с экспоненциальной задержкой: 2, 4, 8 секунд. Каждая попытка логируется в консоль.

**Circuit Breaker** - если 2 запроса подряд завершились ошибкой, цепь «размыкается» на 30 секунд. В этот период новые запросы отклоняются сразу, не тратя время на ожидание таймаута.

**Timeout** - каждый отдельный запрос ограничен 5 секундами. Без этого Retry мог бы ждать бесконечно долго на зависшем соединении.

Порядок важен: Timeout оборачивает каждую попытку, а не весь цикл Retry. Именно поэтому `TimeoutRejectedException` добавлен в список условий Retry.
