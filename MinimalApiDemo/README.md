# MinimalApiDemo

Самый простой способ поднять HTTP-сервер на .NET - без контроллеров, без атрибутов, буквально несколько строк. Этот проект демонстрирует, как работает ASP.NET Core Minimal API.

Два эндпоинта:

- `GET /greet/{name}` - возвращает строку приветствия
- `GET /age/{year}` - считает возраст по году рождения, возвращает JSON-объект

```bash
dotnet run --project MinimalApiDemo
```

```text
GET /greet/Артём     →  Привет, Артём!
GET /age/1995        →  { "birthYear": 1995, "currentAge": 30 }
```

Здесь хорошо видно, как .NET автоматически привязывает `string` и `int` из сегмента URL, и как анонимный объект превращается в JSON без каких-либо дополнительных настроек.
