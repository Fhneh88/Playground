# TodoManager

Простой in-memory менеджер задач для отработки базовой логики работы со списком: добавление, отметка выполнения, вывод всех задач.

Команды читаются из стандартного ввода, состояние живёт только в памяти - при следующем запуске список пустой. Для персистентного хранения см. [TaskCLI](../TaskCLI).

## Формат ввода

```text
<количество команд>
<команда 1>
...
```

Поддерживаемые команды: `add <текст>`, `done <номер>`, `list`.

## Пример

```text
5
add Купить молоко
add Позвонить врачу
done 1
list
done 5
```

```text
added: Купить молоко
added: Позвонить врачу
done: Купить молоко
[x] Купить молоко
[ ] Позвонить врачу
bad index
```

`bad index` - ответ при попытке обратиться к несуществующему номеру, без исключений.
