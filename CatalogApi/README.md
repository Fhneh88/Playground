# CatalogApi

REST API для управления каталогом товаров и категорий. Построен на ASP.NET Core Minimal API - без контроллеров, без атрибутов, всё в одном файле `Program.cs`.

## Эндпоинты

### Категории

```text
GET  /categories                    - список всех категорий
POST /categories                    - создать категорию  { "name": "..." }
```

### Товары

```text
GET    /products                    - все товары с названием категории
POST   /products                    - создать товар  { "name", "price", "categoryId" }
GET    /categories/{id}/products    - товары конкретной категории
PUT    /products/{id}/price         - обновить цену  { "newPrice": ... }
DELETE /products/{id}               - удалить товар
```

## Запуск

```bash
dotnet run --project CatalogApi
```

База данных `catalog.db` создаётся рядом с проектом автоматически при первом старте - никакой дополнительной настройки не нужно.

## Архитектура

Бизнес-логика вынесена в `ProductService`, эндпоинты просто вызывают его методы. Для категорий отдельного сервиса нет - логика простая, поэтому работа с `AppDbContext` идёт прямо из лямбды.

Для несуществующих ресурсов эндпоинты возвращают `404 Not Found`, для успешного создания - `201 Created` с `Location`-заголовком.
