# AgeCalCulator

Первый шаг в C# - консольное приложение, которое спрашивает имя и год рождения и считает возраст.

Ничего сложного, но именно здесь отрабатываются базовые вещи: чтение ввода, валидация, цикл с повтором при ошибке и вычисление через `DateTime.Now.Year`. Результат намеренно говорит «30 или 31 год» - потому что по одному лишь году рождения точный возраст не узнать: нужна ещё дата.

## Запуск

```bash
dotnet run --project AgeCalCulator
```

```text
Введите ваше имя: Артём
Введите год рождения: 1995
Артём, вам сейчас 30 или 31 год(лет) (зависит от того, был ли уже день рождения в этом году).
```

Если ввести не число или год вне диапазона 1900–сегодня - приложение вежливо попросит попробовать ещё раз.
