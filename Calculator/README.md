# Calculator

Минималистичный консольный калькулятор - одна строка ввода, один результат на выходе.

Задача учебная: отработать `switch`-выражение и граничный случай деления на ноль. Программа читает строку вида `<число> <оператор> <число>`, разбирает её и выводит результат. Если делитель равен нулю - печатает `error` и завершается, не бросая исключения.

## Запуск

```bash
dotnet run --project Calculator
```

```text
10 + 5
15

8 / 0
error
```

Поддерживаемые операторы: `+`, `-`, `*`, `/`. Всё остальное выбрасывает `InvalidOperationException` - намеренно, чтобы оставить простоту.
