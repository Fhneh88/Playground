# LogStats

CLI-инструмент для анализа access-логов веб-сервера. Умеет читать как обычные файлы, так и сжатые `.gz` - прямо на лету, без распаковки во временный файл.

## Что выводит

```text
Всего запросов: 1500

Топ IP:
  203.0.113.42    312
  198.51.100.7    201
  ...

Топ URL:
  GET /api/products    450
  GET /                312
  ...

Статус-коды:
  2xx    1200 (80%)
  4xx     250 (17%)
  5xx      50 ( 3%)
```

Топ выводится по 5 записей с выравниванием колонок - ширина подстраивается под самую длинную строку в наборе.

## Использование

```bash
dotnet run -- access.log
dotnet run -- access.log.gz
dotnet run -- access.log --since 2024-01-01 --until 2024-01-31
```

Если файл не найден или не удаётся распарсить строку - инструмент сообщает об этом и продолжает работу, не падая на первой же ошибке. Количество нераспознанных строк выводится отдельно.

## Про декомпрессию

Расширение файла определяется по суффиксу `.gz`. Если он есть - поверх `FileStream` надевается `GZipStream`, и дальнейший код работает с единым `StreamReader` без каких-либо ветвлений.

```csharp
using Stream stream = filePath.EndsWith(".gz", StringComparison.OrdinalIgnoreCase)
    ? new GZipStream(raw, CompressionMode.Decompress)
    : raw;
```
