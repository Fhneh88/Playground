# LogAnalyzer

Консольная задача на разбор лог-файла: прочитать N строк, извлечь IP-адрес из каждой (всё до первого пробела) и вывести топ-3 по количеству запросов.

При одинаковом числе запросов порядок определяется алфавитно - чтобы вывод был детерминированным. Реализовано через `Dictionary` + `OrderByDescending(...).ThenBy(...)`.

## Формат ввода

```text
<количество строк>
<строка лога>
...
```

## Пример

```text
5
192.168.1.1 GET /index.html
10.0.0.1 GET /api
192.168.1.1 POST /login
10.0.0.1 GET /api
192.168.1.1 GET /
```

```text
192.168.1.1 3
10.0.0.1 2
```
