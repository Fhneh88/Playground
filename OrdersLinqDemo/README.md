# OrdersLinqDemo

Четыре LINQ-задачи на реальной таблице заказов в SQLite. Цель - не выучить синтаксис, а почувствовать, как разные операции складываются в читаемые цепочки.

При запуске база создаётся и заполняется тестовыми данными автоматически, так что сразу виден и сам запрос, и его результат.

## Задачи

**1. Оплаченные заказы свыше 1000 руб. от новых к старым**
`Where` + `OrderByDescending` - фильтрация с сортировкой.

**2. Группировка по городу**
`GroupBy` + `Select` + `Sum` - сколько заказов и какая суммарная выручка в каждом городе.

**3. Клиент с наибольшей суммой покупок**
`GroupBy` + `MaxBy` - найти «победителя» по агрегированному полю.

**4. Города с неоплаченными заказами**
`Where` + `Distinct` + `OrderBy` - дедупликация после фильтрации.

## Запуск

```bash
dotnet run --project OrdersLinqDemo
```

Файл `orders.db` создаётся рядом с проектом при первом запуске.
