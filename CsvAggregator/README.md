# CsvAggregator

Учебная задача на парсинг CSV и агрегацию данных: прочитать таблицу, найти нужные столбцы по заголовку и просуммировать `amount` для каждой `category`.

Интересная часть - заголовок не фиксирован. Программа сначала ищет индексы столбцов `category` и `amount` через `Array.IndexOf`, и только потом обходит строки. Это делает код устойчивым к перестановке столбцов во входном файле.

## Формат ввода

```text
<количество строк данных>
<заголовок CSV>
<строки данных>
```

## Пример

```text
4
category,amount
food,100
tech,200
food,50
tech,300
```

Вывод (в алфавитном порядке по категории):

```text
food 150
tech 500
```
